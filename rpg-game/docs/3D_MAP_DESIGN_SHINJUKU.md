# 新宿中央区画 - 3Dマップ設計書

## 🌆 世界観とコンセプト

### 時代設定
- **西暦2050年、東京・新宿**
- AIアーク支配下の完璧に管理された都市
- サイバーパンク × 和の融合
- 無機質で感情のない街並み

### 雰囲気
- **ビジュアル**: ブレードランナー + 攻殻機動隊風
- **カラーパレット**:
  - 基調色: 暗いグレー、深い青
  - アクセント: ネオンブルー、サイバーシアン
  - 禁忌色: 暖色系は極力避ける（人間性の喪失を表現）
- **照明**: 冷たい蛍光灯、散発的なネオンサイン
- **天候**: 小雨または曇天（常に薄暗い）

## 👥 NPC配置

### 一般市民（感情喪失者）
- **配置**: 6-8人
- **行動**: ゆっくり歩行、立ち止まる、無表情
- **見た目**: グレーのスーツ、全員似た服装
- **位置例**:
  - 通りを歩く市民 x3
  - ベンチに座る市民 x2
  - 自販機前で待機 x1
  - ビル入口で立ち止まる x1

### アカリ（幼馴染・重要NPC）
- **初期位置**: (40, 30) 中央やや南の路地
- **見た目**:
  - 髪: 赤みがかった茶髪（珍しい暖色）
  - 服: 白とシアンのツートンコート
  - 目: 生き生きとした表情（他NPCと対照的）
- **イベント**: ゲーム開始直後に会話イベント

### パトロールドローン
- **配置**: 2-3体（上空+地上）
- **見た目**: 球体、青い光のリング、監視カメラ
- **行動**: 一定ルートを巡回、近づくと警告音
- **位置**:
  - 上空 (25, 20) 高さ15m
  - 地上 (10, 35)
  - 出入口付近

## 📊 技術仕様（Blender → Three.js）

### モデリング方針
- **LOD（Level of Detail）**: 3段階
  - LOD0: 近距離（5m以内） - 高ポリゴン
  - LOD1: 中距離（5-20m） - 中ポリゴン
  - LOD2: 遠距離（20m以上） - 低ポリゴン

### ポリゴン数目安
| オブジェクト | LOD0 | LOD1 | LOD2 |
|------------|------|------|------|
| 主要ビル | 5000 | 2000 | 500 |
| 小物 | 500 | 200 | 50 |
| キャラクター | 3000 | 1500 | 300 |
| 全体シーン | ~50k | ~20k | ~5k |

### エクスポート形式
- **3D**: GLTF/GLB (Three.js最適)
- **テクスチャ**: PNG (2048x2048 / 1024x1024)
- **アニメーション**: 組み込み（歩行、待機）

### 最適化
- テクスチャアトラス使用
- インスタンシング（同じオブジェクト）
- オクルージョンカリング
- ベイク済みライティング（静的オブジェクト）
