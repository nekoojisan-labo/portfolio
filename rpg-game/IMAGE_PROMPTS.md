## 推奨画像生成ツール

1. **Midjourney** - 高品質イラスト向け
2. **DALL-E 3** - 一般的な画像生成
3. **Stable Diffusion** - カスタマイズ可能、ローカル実行可
4. **Aseprite** - ピクセルアート手作業用

## 画像サイズ推奨

| 用途 | サイズ | 形式 |
|------|--------|------|
| フィールドアイコン | 32x32 px | PNG (透過) |
| バトル立ち絵 | 256x512 px | PNG (透過) |
| 背景 | 800x600 px | PNG/JPG |
| タイルセット | 32x32 px (個別) | PNG (透過) |
| UIアイコン | 16x16 or 32x32 px | PNG (透過) |
| タイトルロゴ | 800x200 px | PNG (透過) |

## フォルダ構成

```
rpg-game/
└── assets/
    ├── bgm/           - BGM・効果音
    ├── characters/    - キャラクター画像全般（フィールド、立ち絵、敵など）
    │                    例: kaito_field.png, kaito_portrait.png, kaito_battle.png
    │                        akari_field.png, drone_enemy.png, etc.
    ├── icons/         - アイテム・装備アイコン
    │                    例: potion.png, sword.png, armor.png
    ├── maps/          - マップ背景・タイルセット
    │                    例: shinjuku_bg.png, shrine_bg.png
    └── ui/            - UI要素・ロゴ・エフェクト
                         例: title_logo.png, hp_icon.png, slash_effect.png
```
