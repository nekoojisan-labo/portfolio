# 神前御籤

荘厳な神社背景と、実物のおみくじ風の結果画像を使った静的おみくじアプリです。

## 確率

- 大吉: 10%
- 中吉: 20%
- 小吉: 23%
- 吉: 27%
- 末吉: 15%
- 凶: 4%
- 大凶: 1%

## 画像

- `assets/generated/shrine-bg.png`: 生成した神社背景
- `assets/generated/omikuji-paper.png`: 生成したおみくじ用紙
- `assets/fortunes/*.webp`: 7運勢 × 20番の本文入りおみくじ画像

本文データは公開中の既存ページから維持し、表示パターンは各運勢20番に整理しています。
