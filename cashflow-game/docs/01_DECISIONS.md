# マネーアドベンチャー 設計判断書

## 確定した5つの判断

### 1. 脱出判定式
**選択: A（かんたんモード）+ C（ふつう以上）**

| モード | 判定式 |
|--------|--------|
| かんたん | 不労所得 >= 生活費 |
| ふつう | 不労所得 >= 生活費 + 余裕(20%) を **2ターン連続** |
| チャレンジ | 不労所得 >= 生活費 + 緊急資金(3ヶ月分) を **3ターン連続** |

**理由:**
- かんたんモードはシンプルさを優先し、子どもが達成感を得やすくする
- ふつう以上は「安定性」の概念を導入し、一時的な成功ではなく継続的な成功を学ばせる

---

### 2. 協力取引のMVP範囲
**選択: B（貸付 + 共同購入）**

| 取引タイプ | MVP | Phase 2 |
|------------|-----|---------|
| 貸付 | O | O |
| 共同購入 | O | O |
| スキル取引 | - | O |
| 贈与 | - | O |

**理由:**
- 貸付だけでは協力の幅が狭い
- 共同購入は「協力の価値」を最も分かりやすく体験できる
- 贈与・スキル取引は複雑なため Phase 2 に延期

---

### 3. 破綻時の処理
**選択: B（救済あり・リスタート可能）**

```
【破綻フロー】
現金マイナス
    │
    ▼
警告表示 + 選択肢提示
    │
    ├─→ 資産売却で回復
    │
    ├─→ 仲間からの緊急貸付
    │
    └─→ リスタート（ペナルティなし、経験値保持）
```

**理由:**
- 教育目的のゲームで「即脱落」は学習機会の損失
- 「失敗しても立て直せる」というメッセージを伝える
- リスタート時も「経験値」として学びを記録し、ポジティブな体験にする

---

### 4. イベントの発生方式
**選択: B（プレイヤー状態で重み付け）**

| プレイヤー状態 | 出やすくなるカード |
|----------------|-------------------|
| 現金が少ない | 節約チャンス、小さな収入機会 |
| 借金が多い | 返済のコツ、収入アップ機会 |
| 投資経験少 | 小規模投資チャンス |
| 順調 | リスクイベント（学びのため） |
| 脱出間近 | 最終判断を試すイベント |

**理由:**
- 完全ランダムより教育効果が高い
- プレイヤーの状況に応じた「適切な学び」を提供
- ゲームバランスの自動調整効果も期待できる

---

### 5. AI参加の必須性
**選択: B（協調型AIを最低1人必ず入れる）**

```
【ルーム構成】
4人プレイ
    │
    ├─ 人間4人 → AI補助なし（オプションでコーチAI）
    │
    ├─ 人間3人 → AI 1人参加（協調型）
    │
    ├─ 人間2人 → AI 2人参加（協調型 + バランス型）
    │
    └─ 人間1人 → AI 3人参加（協調型 + バランス型 + 積極型）
```

**理由:**
- 協力型ゲームの性質上、最低1人は「協力を促進する」存在が必要
- 人間だけだと協力が発生しにくい可能性がある
- AIが協力のモデルを示すことで、学習効果が高まる
