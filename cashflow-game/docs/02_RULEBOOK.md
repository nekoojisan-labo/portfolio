# マネーアドベンチャー ルールブック

## お金の冒険を通じて、みんなでラットレースを脱出しよう！

---

## 1. ゲームの目的（もくてき）

このゲームの目的は **「全員でラットレースから脱出すること」** です。

### ラットレースとは？
```
┌─────────────────────────────────────────┐
│  🐀 ラットレース                        │
│  ─────────────────────────────          │
│  「働かないとお金がもらえない状態」      │
│                                         │
│  働く → お金をもらう → 使う → また働く  │
│   ↑                           ↓        │
│   └───────────────────────────┘        │
│                                         │
│  このループから抜け出すのが目標！        │
└─────────────────────────────────────────┘
```

### 脱出条件（だっしゅつじょうけん）
```
┌─────────────────────────────────────────┐
│  🎯 脱出するには                        │
│  ─────────────────────────────          │
│                                         │
│  「持っているだけで入るお金」            │
│          ≧                              │
│  「毎月かならず出ていくお金」            │
│                                         │
│  これが達成できたら脱出！                │
└─────────────────────────────────────────┘
```

**大切なこと:** ひとりだけ脱出してもゲームは終わりません。**全員が脱出**してはじめてクリアです！

### プレイ人数
- 1〜4人（足りない人数はAIが参加）

### ゲーム時間の目安
- かんたんモード: 20〜30分
- ふつうモード: 30〜45分

### 最初にもらうもの

| 項目 | かんたんモード |
|------|----------------|
| 💰 持っているお金 | 300 コイン |
| 👔 給料（毎月入る） | 100 コイン |
| 🏠 生活費（毎月出る） | 60 コイン |

## 3. ゲームの進め方

### 1ターン = ゲームの中の1ヶ月

```
┌──────────────────────────────────────────┐
│  🎮 1ターンの流れ                        │
└──────────────────────────────────────────┘

    STEP 1: 収入フェーズ
         │  └─ 💰 お金をもらう
         ▼
    STEP 2: 支出フェーズ
         │  └─ 💸 お金を払う
         ▼
    STEP 3: イベントフェーズ
         │  └─ 🎲 カードを引く
         ▼
    STEP 4: 行動フェーズ
         │  └─ 🤔 買う？売る？借りる？返す？
         ▼
    STEP 5: 協力フェーズ
         │  └─ 🤝 仲間と相談・取引
         ▼
    STEP 6: チェックフェーズ
              └─ ✅ 脱出できた？→ 次の人へ
```

---

## 4. 各フェーズのくわしい説明

### STEP 1: 収入フェーズ 💰

```
もらえるお金 = 給料 + 持っているだけで入るお金

例）給料100 + 家賃収入20 = 120コインもらえる
```

### STEP 2: 支出フェーズ 💸

```
払うお金 = 生活費 + ローン返済

例）生活費60 + 車ローン15 = 75コイン払う
```

### STEP 3: イベントフェーズ 🎲

カードを1枚引きます。3種類のカードがあります。

| カード | 説明 |
|--------|------|
| 💰 チャンスカード | 投資や買い物のチャンス！ |
| 💸 トラブルカード | 急な出費が発生！ |
| 📚 学びカード | スキルアップ！ |

### STEP 4: 行動フェーズ 🤔

以下の行動ができます（しなくてもOK）。

| 行動 | 説明 |
|------|------|
| 🛒 買う | 資産を買う（不動産、株など） |
| 💰 売る | 持っている資産を売る |
| 💳 借りる | お金を借りる（ローン） |
| 💵 返す | 借りたお金を返す |

### STEP 5: 協力フェーズ 🤝

仲間と協力できます！

| 協力の種類 | 説明 |
|------------|------|
| 💰 貸し借り | 仲間にお金を貸す／借りる |
| 🏠 共同購入 | 高い物件をみんなで買う |

### STEP 6: チェックフェーズ ✅

脱出条件を満たしているかチェック！

---

## 5. 協力ルール

### 5-1. お金の貸し借り

```
┌─────────────────────────────────────────┐
│  💰 貸し借りのルール                    │
├─────────────────────────────────────────┤
│                                         │
│  ① 両方が「OK」と言ったら成立           │
│  ② 返済は自動で毎月引かれる             │
│  ③ 金利の上限は10%まで                  │
│  ④ 返済回数は12回まで                   │
│                                         │
│  例）100コインを借りて、毎月25コイン    │
│      × 4回 で返す                       │
│                                         │
└─────────────────────────────────────────┘
```

### 5-2. 共同購入

```
┌─────────────────────────────────────────┐
│  🏠 共同購入のルール                    │
├─────────────────────────────────────────┤
│                                         │
│  ① 2〜4人で一緒に買える                 │
│  ② 出したお金の割合で収入を分ける       │
│  ③ 売るときは全員の「OK」が必要         │
│                                         │
│  例）400コインのアパートを2人で購入     │
│      たろう 200 ／ はなこ 200           │
│      毎月の収入40 → 各20コインずつ      │
│                                         │
└─────────────────────────────────────────┘
```

## 6. 資産と負債（しさん と ふさい）

### 資産 = お金を生むもの 🏠📈

| 資産の例 | 説明 |
|----------|------|
| アパート | 毎月家賃が入ってくる |
| お店 | 毎月売上の一部が入る |
| 株 | 配当がもらえる（上下する） |

### 負債 = お金が出ていくもの 💸

| 負債の例 | 説明 |
|----------|------|
| 車のローン | 毎月返済が必要 |
| 家のローン | 毎月返済が必要 |
