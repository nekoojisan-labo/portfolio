## イベント発生の重み付けルール

プレイヤーの状態に応じてカードの出現率が変わります。

### 重み付けテーブル

| プレイヤー状態 | 出やすいカード | 出にくいカード |
|----------------|----------------|----------------|
| 現金が少ない（生活費2ヶ月分以下） | L05,L06,C04,C13 | T19,C07,C08 |
| 借金が多い（3件以上） | L05,T20,C14 | C01-C08（購入系） |
| 投資経験が少ない（資産0-1件） | C04,C09,C13,L01,L02 | C07,C08,C12 |
| 順調（脱出進捗50%以上） | T15-T18,C08 | L09,L12 |
| 脱出間近（80%以上） | T17,T18,C20 | T19,L20 |
| 直近で協力した | C20,L11 | T09-T14 |

### カード配分比率（基本）

```
チャンスカード: 40% (20種から抽選)
トラブルカード: 30% (20種から抽選)
学びカード:     30% (20種から抽選)
```
