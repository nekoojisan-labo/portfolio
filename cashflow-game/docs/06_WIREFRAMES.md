## 6. アニメーション仕様

### 6.1 基本アニメーション

| 要素 | アニメーション | 時間 |
|------|----------------|------|
| ボタンホバー | scale(1.05) | 0.2s |
| カード表示 | fadeIn + slideUp | 0.3s |
| モーダル表示 | fadeIn + scale | 0.25s |
| コイン増減 | count up/down | 0.5s |
| 進捗バー | width変化 | 0.5s |

### 6.2 ゲームアニメーション

| 要素 | アニメーション | 時間 |
|------|----------------|------|
| サイコロ | 回転 + バウンス | 1.0s |
| 駒移動 | パス沿い移動 | 0.5s/マス |
| カードめくり | flip | 0.4s |
| 脱出演出 | 紙吹雪 + 拡大 | 2.0s |
| ターン切替 | フェード | 0.3s |
